# Contact Management System (CMS)

## 1. Project Title
Contact Management System (CMS)

## 2. Overview of the Project
The Contact Management System is a Python console application that helps users store and manage contact information such as name, age, phone number, and email.  
It supports basic CRUD operations along with search and count features.  
The system uses a modular structure, where each operation is written in a separate Python file for easy maintenance.

## 3. Features
- Create a contact  
- View contact details  
- Update contact  
- Delete contact  
- Search for a contact  
- Count total contacts  
- Menu-driven interface  
- Modular Python code  

## 4. Technologies / Tools Used
- Python 3.x  
- Console Input/Output  
- Modular Programming  
- Dictionaries for data storage
